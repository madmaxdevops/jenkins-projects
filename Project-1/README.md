# 🎯 Summer Camp Registration App

A Django-based web application for registering kids into summer camp programs. The app collects user details and stores them in a MySQL database.

---

## 🛠 Tech Stack

* Python
* Django
* MySQL
* HTML/CSS

---

## 📌 Features

* Registration form with:

  * Full Name
  * Age
  * Gender
  * Father Name
  * Mobile Number
  * Location
* Data stored in MySQL
* Success message after submission

---

## ⚙️ Setup & Run

```bash
git clone <your-repo-url>
cd summercamp-registration-python-django
pip install -r requirements.txt
```

### Run Migrations

```bash
python3 manage.py makemigrations
python3 manage.py migrate
```

### Start Server

```bash
python3 manage.py runserver
```
