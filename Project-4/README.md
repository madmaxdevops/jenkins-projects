# 🚀 AngularJS Sample Web App (Seed Project)

A starter template for building **AngularJS web applications**, preconfigured with development tools, testing setup, and project structure for rapid development.

---

## 📌 Overview

This project provides:

* Preconfigured AngularJS application structure
* Sample controllers and views
* Integrated unit & end-to-end testing setup
* Development server for quick startup

---

## 🛠 Tech Stack

* AngularJS
* Node.js & npm
* Jasmine (Unit Testing)
* Karma (Test Runner)
* Protractor (E2E Testing)

---

## ⚙️ Prerequisites

* Node.js & npm
* Git
* Java JDK (for E2E testing)

---

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/angular/angular-seed.git
cd angular-seed
```

---

### Install Dependencies

```bash
npm install
```

---

## ▶️ Run Application

```bash
npm start
```

Open in browser:

http://localhost:8000/index.html

---

## 🧪 Testing

### Run Unit Tests

```bash
npm test
```

### Run Single Test Execution

```bash
npm run test-single-run
```

### Run End-to-End Tests

```bash
npm run protractor
```
