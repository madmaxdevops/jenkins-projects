# 🚀 Simple Java Maven App

A lightweight Java application built using **Maven**, designed to demonstrate a complete **CI/CD pipeline using Jenkins**.

---

## 📌 Overview

This project showcases:

* Building a Java application using Maven
* Running unit tests
* Generating build artifacts (`.jar`)
* Automating build and deployment using Jenkins

---

## 🛠 Tech Stack

* Java
* Maven
* Jenkins
* Git & GitHub

---

## 📂 Project Structure

```
simple-java-maven-app/
├── src/
│   ├── main/java/
│   └── test/java/
├── pom.xml
└── Jenkinsfile
```

---

## ⚙️ Build & Test

```
mvn clean package
```

---

## ▶️ Run Application

```
java -jar target/*.jar
```

---

## 🔄 CI/CD Pipeline (Jenkins)

* Checkout code
* Build with Maven
* Run tests
* Package artifact
* Deploy (optional)
